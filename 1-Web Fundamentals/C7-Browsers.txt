
Browsers :-

      - It's a application Software, which acts as a client that communicates with web servers.

      - It retrieves web pages, processes content, and present it to user in a visually appealing and interactive format.

      -The entire process involves multiple protocols, technologies, and layers of abstraction to provide a seamless and secure browsing experience.

Working :-
  
  HTTPS Request   ->    DNS Resolution    ->    Server Communication    ->     HTML Rendering    ->     CSS Styling     ->
  
  Javascript Execution     ->     Rendering the Page      ->      User Interaction.
